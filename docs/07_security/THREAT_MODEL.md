# Модель угроз (Threat Model) для ChatAVG v2.3

**Дата:** 7 мая 2026

Данная модель угроз определяет ключевые векторы атак на архитектуру v2.3 (Meaning-first Agent Execution Platform) и закладывает требования для Policy/Cost Control Plane.

## 1. Защищаемые активы (Assets)
1. **Ключи API провайдеров** (OpenAI, LiteLLM, E2B, базы данных).
2. **Данные пользователей** (история миссий, извлеченные смыслы/Claims, загруженные файлы).
3. **Бюджет** (защита от исчерпания лимитов токенов/вызовов).
4. **Инфраструктура сервера** (защита от удаленного выполнения кода - RCE).

## 2. Поверхности атаки (Attack Surfaces) и Митигация

| Вектор атаки | Описание | Мера защиты (Mitigation) |
| :--- | :--- | :--- |
| **Prompt Injection / Jailbreak** | Пользователь заставляет ИИ игнорировать инструкции и выдать системный промпт или совершить вредоносное действие. | Использование "Approval Requests" (запрос разрешения у человека на любые ToolCalls), языковые Guardrails в Adequacy Engine. |
| **SSRF (Server-Side Request Forgery)** | Агент или MCP Tool обращается к внутренним IP-адресам сервера (127.0.0.1, localhost). | Egress-фильтрация в песочницах. Запрет на обращение к локальным ресурсам из инструментов поиска/выполнения. |
| **Утечка секретов (Secrets Leakage)** | API ключи случайно передаются в контекст LLM или выводятся в консоль. | Хранение секретов строго в `.env`. Секреты НЕ передаются в промпты, а инжектятся на уровне адаптеров (Model Gateway). Redaction логов. |
| **Исчерпание бюджета (Cost Attack)** | Злоумышленник (или зацикленный агент) запускает бесконечный цикл, тратящий бюджет на LLM. | Cost Control Plane: установка жестких лимитов на Tenant/AgentRun. Прерывание Temporal workflow при превышении бюджета. |
| **Побег из песочницы (Sandbox Escape)** | Вредоносный код, сгенерированный ИИ в режиме Forge, пытается заразить хост. | Полная изоляция: использование сторонних microVM (E2B / Daytona). Запрет `eval()` локально на сервере платформы. |

## 3. Политика "Default Deny" (Запрет по умолчанию)
Все опасные инструменты (запись в файл, изменение настроек, отправка сетевых запросов) по умолчанию отключены или требуют явного одобрения (Human-in-the-loop Approval) через интерфейс Mission Room.
