# Product Review: ER-Layer Usefulness

## Дата: 2026-05-07
## Статус: Утверждено для MVP

### Оценка ценности Semantic Layer
ER Meaning Layer был задуман как главное конкурентное преимущество (USP) ChatAVG v2.3, предлагающее пользователю не просто текст ответа, а структуру смысловых утверждений (claims), их проверку на границы применимости (domain boundaries) и уверенность (strength).

### Ключевые выводы по MVP:
1. **Прозрачность вывода:** `ClaimExtractor` успешно работает. Модель вычленяет из ответов факты, гипотезы и вопросы. Это позволяет фронтенду строить интерфейсы (например, Conflict Cards), где пользователь видит "что именно" утверждает нейросеть.
2. **Адекватность ответов (Adequacy Covenant):** Правила `DomainBoundary` (Medical, Legal, Psychological) автоматически понижают уровень уверенности (например, с "fact" на "weak_hypothesis"). Это решает фундаментальную продуктовую проблему — ИИ перестает звучать самоуверенно там, где он не эксперт.
3. **Безопасность (No Hidden Authority):** Блокировка психодиагностики дает пользователям чувство приватности. Система не делает выводов об их внутреннем мире.

### Пользовательский опыт (UX):
В настоящее время извлеченные утверждения собираются в `ClaimLedger`, а на бэкенде доступны через API MVP Dashboard. 
**Следующий шаг:** Вывести UI-часть (Role Passes, Artifact Workspace) в Спринте 12, чтобы пользователь мог визуально взаимодействовать с ER-слоем.

**Резолюция:** Слой ER полезен. Semantic PoC успешен. Разрешен переход к Спринтам 10+ для интеграции RAG с семантикой.
