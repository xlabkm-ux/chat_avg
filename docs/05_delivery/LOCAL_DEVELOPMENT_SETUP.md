# Локальная настройка среды (Local Development Setup)

Данный документ описывает процесс развертывания проекта ChatAVG v2.3 на локальной машине разработчика.

## 1. Требования (Prerequisites)
*   **Node.js**: v20.x или новее.
*   **npm**: v10.x или новее.
*   **Git**
*   *Опционально:* Docker Desktop (для запуска локального Temporal, LiteLLM Proxy или баз данных).

## 2. Инициализация проекта
```bash
# 1. Переход в рабочую папку
cd c:\AG\agsys

# 2. Установка зависимостей
npm install

# 3. Настройка переменных окружения
# Скопируйте .env.example в .env и заполните необходимые ключи
# (Секреты не коммитятся в Git!)
```

## 3. Базовые переменные окружения (.env)
Минимальный набор для запуска базового чата (Fast Path):
```env
PORT=3000
NODE_ENV=development

# Ключи для Model Gateway
OPENAI_API_KEY=sk-... 

# Ключи для Forge (потребуются позже в Sprint 14)
# E2B_API_KEY=e2b_...
```

## 4. Запуск проекта
**Запуск основного сервера:**
```bash
npm run dev
# или
node server.js
```

**Обновление карты проекта (ОБЯЗАТЕЛЬНО):**
Перед каждым коммитом разработчик (или AI-агент) обязан обновить карту зависимостей, чтобы документация не отставала от кода:
```bash
node dev_studio/refresh.js
```

## 5. Структура проекта для разработчика
*   **`/docs`** — спецификации, контракты, архитектура. Обязательно к прочтению (особенно `02_architecture` и `04_specs`) перед написанием кода.
*   **`/cons/chatavg`** — текущее ядро системы.
*   **`/dev_studio`** — инструменты анализа кодовой базы (утилиты разработчика).
