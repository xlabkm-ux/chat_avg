# 🔍 UI & Security Audit — Chat AVG

Этот файл является единым журналом всех проверок качества, безопасности и интерфейса проекта.

## 📊 Текущий статус (Итерация Май 2026)
**Общая оценка: 9.5 / 10**

### 🟢 Решенные критические проблемы (Май 2026)
1. **Мобильная адаптация**: Внедрена система media-queries (Base, SM, MD, LG, XL). Поддержка всех устройств.
2. **Safe-area (iOS)**: Использование `env(safe-area-inset-*)` во всех ключевых элементах (Header, Input, Sidebar).
3. **Touch-targets**: Все интерактивные элементы имеют минимальный размер `44x44px` для удобства нажатия пальцем.
4. **XSS через Markdown**: Внедрена санитизация через `DOMPurify` во всех местах рендеринга сообщений.
5. **Logout**: Полноценный выход с очисткой токенов и состояния реализован и проверен.

### 🟡 Средняя критичность (В ОЧЕРЕДИ)
1. **Отсутствие темной/светлой темы**: Интерфейс заблокирован в темном режиме.
2. **Индикация статуса**: `status-dot` не всегда корректно отражает состояние llama-server.

## 🕒 История аудитов и решенных проблем

### Аудит Безопасности и Архитектуры (Май 2026)
- [x] **Race Conditions**: Решено переходом на SQLite (атомарные транзакции).
- [x] **Хеширование паролей**: Заменено на `bcryptjs` (ранее был небезопасный SHA-256 без соли).
- [x] **Path Traversal**: Внедрена очистка `username` через `path.basename()`.
- [x] **SSRF Protection**: Добавлена блокировка запросов к `localhost` и приватным IP-диапазонам для внешних API.

### Аудит MimikaStudio (Март 2026)
- [x] **Windows Compatibility**: Исправлены жестко заданные пути `/tmp`.
- [x] **Unicode Handling**: Решена проблема "битой" кириллицы (`?????`) через форсирование UTF-8.
- [x] **Async/Sync Conflicts**: Исправлена блокировка Event Loop при загрузке тяжелых моделей Whisper.

### Прагматичный пересмотр (Критика Enterprise-подхода)
- [x] **Упрощение развертывания**: Отказ от нативных C++ зависимостей (типа `bcrypt`) в пользу JS-аналогов для легкой установки на Windows.
- [x] **Hot-reload**: Сохранение возможности прямой правки конфигураций без перезагрузки сервера.

## 📐 Действующие стандарты (Guidelines)

### Интерфейс
- Кнопки без текста: `aria-label` обязателен.
- Минимальный размер кликабельной зоны: `44px x 44px`.
- Сетка: Использовать CSS-переменные `--sp-*` для отступов.

### Безопасность
- Весь ввод пользователя: Санитизация через `DOMPurify`.
- Пароли: Минимум 8 символов, хеширование `bcrypt`.
- API: Валидация всех входящих JSON через схемы `zod`.
