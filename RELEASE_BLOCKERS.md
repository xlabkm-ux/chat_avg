# 🚫 RELEASE_BLOCKERS.md

Список критических проблем, блокирующих релиз ChatAVG v2.3 в эксплуатацию.

## P0 — Критические (Runtime & Security)
- [ ] **RB-001: Sandbox Local Fallback in Prod.** Текущий SandboxManager позволяет использовать `LocalAdapter` в продакшене, что небезопасно. Необходимо реализовать fail-closed поведение.
- [ ] **RB-002: Mock Temporal Activities.** Большинство активностей Temporal являются заглушками. Система не обеспечивает реальную устойчивость исполнения.
- [ ] **RB-003: Lack of Idempotency.** Отсутствует механизм ключей идемпотентности для инструментов и ранов, что может привести к дублированию побочных эффектов.
- [ ] **RB-004: Insecure Admin Boot.** Генерация пароля админа в логах продакшена и отсутствие требования обязательного пароля из ENV.

## P1 — Высокие (Functional & Data)
- [ ] **RB-005: Mock Knowledge Retrieval.** RAG-система работает на моках. Реальный поиск по знаниям отсутствует.
- [ ] **RB-006: Ephemeral Claim Ledger.** Семантические утверждения не сохраняются надежно, что делает невозможным их аудит и использование в долгоживущих миссиях.
- [ ] **RB-007: Dashboard Placeholders.** Метрики на панели управления (latency, quality) являются захардкоженными значениями.

## P2 — Средние (UX & Docs)
- [ ] **RB-008: Misleading Backlog.** Документация заявляет о готовности компонентов, которые фактически являются скелетами.
- [ ] **RB-009: Missing Migration Path.** Отсутствует проверенный план миграции с V1 на V2.3 с возможностью отката.
